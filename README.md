# ☕ Yega Coffee Service Website

A responsive website built for a coffee service business, showcasing modern design and functional backend features.

### 🛠️ Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (For processing contact forms and orders)

### 🚀 Features
* **Modern UI:** Clean, responsive design optimized for mobile and desktop users.
* **Interactive Elements:** Dynamic client-side features powered by JavaScript.
* **Contact & Order System:** A secure backend PHP script (`contact.php`) that handles user messages and service inquiries.

### 📂 Project Structure
* index.html - Main landing page architecture.
* style.css - Visual presentation and styling rules.
* script.js - Interactive functionality.
* contact.php - Server-side data processing script.
